QuickAid

For the 2018 MinneHack, my team and I created QuickAid - a chatbot that will work with a user to diagnose sports injuries and suggest first aid. The project was originially hosted on AWS and uses a MEAN stack (MongoDB, ExpressJS, AngularJS, and NodeJS).

The sourcecode for the full project can be found here: QuickAid Sourcecode
